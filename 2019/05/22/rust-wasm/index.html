构建包

运行wasm-pack build帮我们做了一系列的事情：

  1. 将rust编译成WebAssembly
  2. 在生成的WebAssembly上运行wasm-bindgen, 从而产生一个包装了WebAssembly的npm模块
  3. 创建了一个pkg文件夹，里面包含了生成的.wasm和.js文件
  4. 根据Cargo.toml在pkg下生成对应的package.json文件
  5. 拷贝 README.md (如果存在) 到pkg下

发布包到npm

到这步以后，我们就可以在npm上看到刚刚发布的包了，如果你在build的时候指定了scope, 那么包名就是@my_npm_username/hello_wasm
